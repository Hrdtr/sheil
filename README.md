# Sheil

Open-source, cross-platform SSH/telnet/serial client — a Termius alternative.

## Requirements

- Node.js (via nub)
- Rust

## Setup

```bash
nub install
```

## Development

```bash
nub run tauri dev
```

## Build

```bash
nub run tauri build
```
